# PalindromeCheckerApp
* Checks whether a word, phrase, or sentence is a palindrome
* Ignores spaces, punctuation, and capitalization
* Real-time validation as the user types
* Highlights matching and non-matching characters
* Displays clear “Palindrome” or “Not a Palindrome” result
* Shows reversed version of the input
* Supports numbers and alphanumeric strings
* One-tap clear/reset button
* Copy result to clipboard option
* Lightweight and fast processing
* Dark mode / light mode support
* Mobile-friendly responsive design
* Error handling for empty input
* Optional history of previous checks
